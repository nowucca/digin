# Project Brief: DigIn

## What It Is
DigIn is a CLI tool that scrapes LinkedIn saved posts, clusters them by topic using ML, and visualizes/exports the results.

## Project Info
- **Developer:** Solo — Steven Atkinson
- **Repo:** github.com/nowucca/digin (public)
- **License:** MIT

## Core Goals
- Automate extraction of LinkedIn saved posts
- Cluster by topic using local ML (no cloud APIs required)
- Visualize with interactive HTML (treemap, scatter, network)
- Export to CSV, JSON, Markdown
- Agent-friendly: --json flag, schema command, Claude Code plugin
